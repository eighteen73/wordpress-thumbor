## Configuration

```php
define( 'THUMBOR_URL', 'https://media.example.com' );
define( 'THUMBOR_SECRET', 'your_thumbor_secret' );
```

If you Thumbor server runs in unsafe mode (highly discouraged) you may set `THUMBOR_SECRET` to `null`.
